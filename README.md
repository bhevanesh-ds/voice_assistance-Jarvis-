# 🤖 Jarvis - Voice Assistant

A Python voice assistant built by Bhevanesh.

## Features
- Tell time & date
- Weather updates
- Tell jokes
- Web search
- Open & close apps
- Play music
- Battery status
- Screenshot
- News headlines

## Setup
pip install speechrecognition pywin32 requests psutil pyautogui pillow

## Run
python main.py
